üöÄ Quick Start

Installation:

git clone https://github.com/pdubbbbbs/claude-code-server.git
cd claude-code-server
pip install -r requirements.txt

Configuration:

cp .env.example .env
# Edit .env and add your ANTHROPIC_API_KEY

Run Server:

python server.py
# Server runs on http://localhost:8002

üì° API Endpoints

POST /execute

Execute code using Claude AI

curl -X POST http://localhost:8002/execute \
  -H "Content-Type: application/json" \
  -d '{"code": "print(2+2)", "language": "python"}'

POST /chat

Chat with Claude about code

curl -X POST http://localhost:8002/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Explain async/await in Python"}'

POST /analyze

Analyze code for bugs and improvements

curl -X POST http://localhost:8002/analyze \
  -H "Content-Type: application/json" \
  -d '{"code": "def add(a,b): return a+b", "language": "python"}'

‚öôÔ∏è Configuration

Configure via environment variables or .env file:

  • ANTHROPIC_API_KEY - Your Claude API key (required)
  • PORT - Server port (default: 8002)
  • RATE_LIMIT - Rate limit per IP (default: 100/minute)
  • CLAUDE_MODEL - Claude model to use (default: claude-3-5-sonnet-20241022)
  • MAX_TOKENS - Max tokens per request (default: 4096)
